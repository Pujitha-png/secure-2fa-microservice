# Secure 2FA MiceoService 

## Description
This project is a secure two-factor authentication (2FA) microservice that generates and validates TOTP codes for user authentication.

## Features
- Generate TOTP codes using a secret key
- Validate user-provided TOTP codes
- Works with the instructor API for seed generation
- Easy to integrate with other applications
